# Phase C Tickets (Codex Wave)

## C1 Command Palette Expansion
- Global command palette with search by workspace title, repo, branch, PR #.
- Actions: next unread, next needs attention, open repo/workspace settings.

## C2 Workspace History / Archive Center
- History browser for workspaces with filters.
- Archive/unarchive workflows and metadata timeline.

## C3 Manual Mode File Editor
- Built-in file editor pane with syntax highlighting.
- Find (⌘F), save, and immediate git diff refresh.

## C4 Performance Pass
- Reduce polling frequency and duplicate refreshes.
- Virtualize heavy lists/diffs where feasible.
- Add lightweight perf counters (render/update timing).

## C5 Shortcut + Accessibility Pass
- Add/validate comprehensive shortcuts.
- Accessibility labels/roles for major controls.
- Shortcut cheat sheet panel (⌘/).

## C6 Integration + Stabilization
- Shared models/router/state for C1–C5.
- Migration additions if needed.
- docs/phase-c-integration.md + tests.
